博学谷 第2天

  2. 切换分支

  3. 删除分支

  4. 合并分支

  5. 后悔药

远程仓库

共享版本库（源代码）

  1. 创建

  2. 给远程仓库中推送内容

  3. 获取远程仓库中所有内容

  4. 获取远程仓库中更新了的内容

  5. 注意 在push之前一定要先pull

远程地址的别名设置

  1. 添加别名

  2. 查看所有别名

  3. 删除别名

  4. 重命名别名

  5. 获取别名对应的地址

  6. 设置别名对应的地址

github的免密登录设置

  1. 生成秘钥对（一个电脑执行一次就行）

  2. 在github的设置中 添加一个ssh key

  3. key中的内容就是将生成的公钥内容粘贴进去就可以了

  4. 以后使用github的时候，直接（只有）使用ssh连接就可以实现免密

github静态页面

  1. 在github中创建一个仓库
  2. 仓库的名字必须是 用户名.github.io
  3. 将写好的静态页面push到这个cnagku
  4. 通过 用户名.github.io 这个地址就可以访问网站了

注意： 一个账户只能有一个这样的项目！！！
